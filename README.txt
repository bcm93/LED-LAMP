Documentación

Este proyecto trata de una pelota luminosa con distintos modos de juego controlados a
partir de una App para Android. 

Para el diseño de la PCB todos los ficheros necesarios se encuentran en la carpeta ball lamp.

Para la alimentación se utilizan 3 pilas AA pero se podría ampliar este proyecto incluyendo una 
bateria, una red de carga y unos módulos de carga inductiva.Para ello, se necesitaría una base 
para el módulo transmisor con una red de adaptación para la corriente. El módulo receptor 
se encontraría dentro de la pelota junto con la batería y el circuito de carga.

El archivo BallLampApp.apk es la app Android que controla la bola, está lista para ser instalada.
IMPORTANTE: necesario emparejamiento previo al juego del Bluetooth con el dispositivo que 
	        se vaya a utilizar.
	        En cada modo de juego es necesario conectar el bluetooth, para ello se debe 
	        pulsar en el titulo de dicha pantalla. AL SALIR DE LA PANTALLA EL BLUETOOTH
	        SE DESCONECTA.


----------------------------------------------------------------------------------------------------------------------------------------------------------------

Modos de juego

Lámpara: luz en varios colores

Mascotas: cambios de luz controlados al pulsar los botones

Patata Caliente: cambios de luz a partir de los giros de la pelota. Cuando falta poco para que 
		finalice el juego la pelota empieza a parpadear. Al llegar al tiempo límite
		la pelota se pone roja.

DOCUMENTATION

This project consists of a bright ball with different game modes which can be controlled from
an Android App.

All the necessary files for the design of the PCB are located in the directory "ball lamp".

Three AA batteries are used as energy supply, but the project could be extended to include a
rechargeable battery, a charging network, and inductive charging modules. 
For this, a base for the transmitter module would be needed together with an adapting 
network for the current. The receiver module would be located inside the ball along with the
battery and the charging circuit.

The file named "BallLampApp.apk" is the Android App which controls the ball and it is ready 
to be installed.

IMPORTANT NOTE: It is necessary to previously sincronize the ball with the Bluetooth device 
		which will be used.
		For each game mode, it is necessary to reconnect the bluetooth by clicking 
		on the title of the screen. WHEN YOU LEAVE THE SCREEN, THE BLUETOOTH 
		CONNEXION AUTOMATICALLY DISCONNECTS.

----------------------------------------------------------------------------------------------------------------------------------------------------------------

GAME MODES

"Lámpara" (Lamp): light available in several colors

"Mascotas" (Pets): light changes can be controlled by pressing the buttons

"Patata Caliente": the light changes when the ball rotates. When the game is about to end, 
		the ball starts flashing and it becomes red when the time limit is reached.
